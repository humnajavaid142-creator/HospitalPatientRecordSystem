                            HOSPITAL PATIENT RECORD SYSTEM
About:
    This is a C++ based program on patient records in hospitals.
    It contains following features:
        i. Add new patient record.
        ii. View all patient records.
        iii. Search patients records (either by name or I'D).
        iv. Store recordss for future.
    -It can help in not wasting too much of time of the patient along with the staff's to do everything manually, everthing is just a click away.
    -It also involve file handling for storing data.

How to RUN:
    - Go to VS Code terminal.
    -Compile: 
         g++ "hospital patient record system" -o hospital.exe 
    -Run:
         ./hospital.exe
